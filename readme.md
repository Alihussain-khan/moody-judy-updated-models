### Welcome to Moody judy Prototype

Its a very basic project for emotion analysis from audio. Its a fastapi backend and react frontend. Both folders have their own readme to get you upto speed.

Its is multimodel platform and requires paid api-key by claude. You dont need to choose models default should be okay. 

##### models
- whisper for transcripts
- emotion2vec+ for Speech emotion recognition
- Beats and Clap for environment acoustics
- claude for reasoning
